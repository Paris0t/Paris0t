Security analyst building tools to automate the tedious parts of defense. Currently doing vulnerability management and RMF compliance for DoD systems — and writing Python to make it all faster.

### What I'm Working On

- **[JobSentry](https://github.com/Paris0t/JobSentry)** — AI-powered job discovery CLI that scrapes job boards, scores listings with Claude AI, and sends daily email/Telegram digests. Built with Python, Playwright, SQLAlchemy, and the Anthropic SDK.
- **[SOAR-EDR Lab](https://github.com/Paris0t/SOAR-EDR-Project)** — Detection and response lab with LimaCharlie (EDR), Tines (SOAR), and TheHive for automated incident triage workflows.
- **Homelab** — Proxmox environment running Pi-hole/Unbound, Vaultwarden with YubiKey FIDO2, Tailscale mesh VPN, Uptime Kuma, and Grafana dashboards. All containerized via Docker/Portainer.
